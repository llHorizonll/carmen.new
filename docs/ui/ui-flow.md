# UI Flow

## Primary Flow
1. Login
2. Select tenant
3. Land on dashboard
4. Navigate into accounting modules
5. Open detail panels or table rows
6. Review approvals, notifications, and audit logs

## Shell Behavior
- compact sidebar groups modules by function
- sticky topbar exposes search, tenant, property, and sync status
- realtime drawer shows notifications and connection state

## Screen Expectations
- dense operational screens
- table-heavy modules
- minimal marketing treatment
- tablet support is required
