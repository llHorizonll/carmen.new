# Financial Table Standards

## Requirements
- sticky headers
- compact rows
- dense layout
- right-aligned amounts
- monospace financial numbers
- summary rows
- column filters
- keyboard-friendly workflows

## Mantine React Table
Use compact density by default.

## Amount Columns
- right aligned
- tabular font
- negative values visually distinct

## UX
- hover states
- fast scrolling
- row click navigation
- export buttons
