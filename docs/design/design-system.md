# Design System

## Visual Philosophy
- premium fintech SaaS
- dark mode first
- dense but readable
- operational finance-focused
- minimal but powerful

## Inspiration
- Linear
- Stripe Dashboard
- Mercury Banking
- Ramp
- Arc Browser
- Vercel

## Density Rules
- compact layouts
- avoid giant whitespace
- small but readable labels
- dense tables
- high information density

## Typography
- strong hierarchy
- tight spacing
- monospace numbers
- right-aligned amounts

## Colors
- graphite backgrounds
- soft layered panels
- thin borders
- restrained accents

## Motion
- subtle Framer Motion transitions
- lightweight notifications
- minimal page transitions

## Anti-Patterns
- no bootstrap look
- no giant KPI cards
- no oversized sidebar
- no toy UI
- no giant border radius

## Readability Rule
Readability is more important than visual effects.

Avoid:
- excessive glow
- low-opacity text
- blurry overlays
- neon cyberpunk aesthetics

Financial UI must remain readable at a glance in operational environments.

Design for:
- finance operators
- accountants
- controllers
- month-end close workflows

Not for gaming aesthetics.
