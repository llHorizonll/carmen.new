# Blueprint

## Product
An internal hotel accounting SaaS platform for hospitality finance teams.

## Positioning
The UI should feel like Bloomberg Terminal meets Linear for hotel finance teams.

## Core Modules
- authentication
- tenant selection
- dashboard
- general ledger
- accounts payable
- accounts receivable
- income audit
- reports
- approvals
- audit logs
- settings
- user management

## Build Rules
- multi-tenant by default
- realtime-enabled
- dark mode first
- dense but readable
- production-oriented, not prototype-oriented

## Delivery Shape
- React SPA
- static build
- IIS hosting
- PocketBase backend
- websocket realtime subscriptions
