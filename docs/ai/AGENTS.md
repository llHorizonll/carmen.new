# AI Agent Rules

Use the docs index in `../README.md` and the category docs under `docs/` as the source of truth.

## Design Rules
Always prioritize:
- density
- readability
- operational workflows
- professional finance aesthetics

## Never Generate
- generic admin templates
- bootstrap-looking layouts
- giant whitespace sections
- decorative dashboards

## Tables
Financial tables are the core UX.
They must feel enterprise-grade and operational.

## Dashboard
Dashboard must feel realtime and alive:
- live indicators
- operational metrics
- financial summaries
- approval queues
- activity feed

## Architecture
Frontend and backend must be production-oriented, not prototype-oriented.

## Goal
Build a production-grade hotel accounting platform.
